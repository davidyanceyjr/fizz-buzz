# cli-ux-review

Purpose
-------
Improve CLI behavior for humans and scripts.

Procedure
---------
1. Inspect help output, subcommands, flags, defaults, exit codes, and stdout/stderr behavior.
2. Check consistency of flag names and semantics.
3. Prefer errors that explain action and remedy.
4. Ensure machine-readable output is not polluted by chatter.
5. Check edge cases like empty input and invalid option.
