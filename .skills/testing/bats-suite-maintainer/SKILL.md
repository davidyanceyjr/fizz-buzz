# bats-suite-maintainer

Purpose
-------
Keep a BATS suite clear, DRY, and reliable.

Procedure
---------
1. Inspect suite structure.
2. Identify duplicated setup and teardown logic.
3. Extract shared helpers where useful.
4. Standardize assertion style and naming.
5. Remove nondeterministic behavior.
6. Improve fixture clarity and execution speed without hiding intent.
