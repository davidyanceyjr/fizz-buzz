# workflow-helpers

Purpose
-------
Provide focused GitHub CLI procedures.

Procedure
---------
Use:
- `gh issue list`
- `gh issue view <number>`
- `gh issue create`
- `gh pr list`
- `gh pr view <number>`
- `gh pr diff`
- `gh pr create`
- `gh pr merge --squash`
- `gh release create <tag>`

Always pair command suggestions with why the command is the right next step.
