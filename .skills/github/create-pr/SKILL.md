# create-pr

Purpose
-------
Prepare a clean pull request summary from a diff and the associated issue.

Procedure
---------
1. Read the issue/spec summary and the diff.
2. Write PR sections: Problem, Approach, Testing, Notes/Tradeoffs.
3. Keep claims grounded in the diff and test evidence.
4. Highlight known limitations honestly.
5. Provide a `gh pr create` command or body-file content.
