# session-handoff

Purpose
-------
Capture working state so future-you or the next AI session can resume quickly.

Procedure
---------
1. Record current branch, active issue, current stage, completed work, remaining work, blockers, and commands/tests to rerun.
2. Update `SESSION.md`.
3. Keep notes short and factual.
4. Do not replace the issue/spec/design with session notes.
